# Description
This repos contains all the work of Planetside 2 Hacking.
It is a .dll made in C++ that you can inject into the game.

# Features
- Skeleton Aimbot
- Aim Prediction
- Speed Hack
- JetPack Velocity Hack
- NoClip / Fly Hack
- ESP Hack (player name, distance)
